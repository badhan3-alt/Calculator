Calculator Project
Overview

This is a simple Java Calculator application built using Java Swing.
It performs basic arithmetic operations like addition, subtraction, multiplication, and division.

Features

Graphical User Interface (GUI) using Java Swing

Supports basic arithmetic operations: +, -, *, /

Lightweight and easy to use

Can be extended for more advanced calculations

Technologies Used

Java

Java Swing (for GUI)

PostgreSQL (optional, if storing calculations)

Docker (optional, if you containerize the app)

(You can remove PostgreSQL/Docker if not used in this project.)

Prerequisites

Java JDK installed (version 8 or above)

Optional: IDE like VS Code, NetBeans, or IntelliJ IDEA
